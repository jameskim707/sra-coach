# 📐 S.R.A 사례 검색란 구조 설계
### v1.2 이후 구현 옵션 (현재는 설계만)

> ⚠️ 본 문서는 **구현용이 아닌 설계 문서**입니다.
> 향후 확장 시 참고용으로 작성되었습니다.

---

## 🎯 목적

정책 담당자 / 평생교육사가 **에듀테크 실패 사례**를 빠르게 검색하고,
**S.R.A의 차별점**을 즉시 확인할 수 있는 기능.

---

## 📦 기능 구조 (설계안)

```
┌─────────────────────────────────────────────┐
│  🔍 사례 검색 (Case Search)                  │
├─────────────────────────────────────────────┤
│                                             │
│  [실패 유형 선택]  ▼                         │
│  ┌─────────────────────────────────────┐   │
│  │ ○ 역할 연결 부재                      │   │
│  │ ○ 인간 개입 상실                      │   │
│  │ ○ 평가 시험 편향                      │   │
│  │ ○ 교사 비참여                         │   │
│  │ ○ 지속성 붕괴                         │   │
│  └─────────────────────────────────────┘   │
│                                             │
│  [지역 선택]  ▼                              │
│  ┌─────────────────────────────────────┐   │
│  │ ○ 국내                                │   │
│  │ ○ 해외                                │   │
│  │ ○ 전체                                │   │
│  └─────────────────────────────────────┘   │
│                                             │
│  [검색] 버튼                                 │
│                                             │
└─────────────────────────────────────────────┘
```

## 📊 출력 형식 (설계안)

```
┌─────────────────────────────────────────────┐
│  📋 검색 결과                                │
├─────────────────────────────────────────────┤
│                                             │
│  선택: 역할 연결 부재 / 국내                  │

│  │ 📌 사례: 코스웨어 AIDT                 │   │

│  │ 지역 역할 제안: 0%                     │   │
│  │                                       │   │
│  │ 💡 S.R.A 해결 방식:                    │   │
│  │ 학습 → 지역 역할 제안 → 실전 참여       │   │
│  └─────────────────────────────────────┘   │
│                                             │
└─────────────────────────────────────────────┘
```

## 🔧 구현 시 고려사항

| 항목 | 내용 |
|------|------|
| **위치** | 사이드바 또는 별도 탭 |
| **접근 권한** | 관리자/교육사 전용 (일반 사용자 비노출) |
| **데이터 확장** | JSON/DB로 분리하여 사례 추가 용이하게 |
| **UX** | 검색 결과 + S.R.A 해결 방식 병기 |
